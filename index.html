Projects

Selected research
and engineering work

01 / FYP

Biomass-Derived Carbon Filter for Turbid Water Treatment

Designed and fabricated a low-cost water filtration system using rice husk ash (RHA) and Moringa oleifera as natural coagulants. Employed Response Surface Methodology with a Face-Centered Central Composite Design (CCD) in Design Expert to optimise coagulant dosage and contact time.

Final Year Project RSM / CCD Design Expert Moringa oleifera Rice Husk Ash Kinetic Modelling
02 / ML

Machine Learning for Froth Flotation Optimisation

Developed a three-tier LOOCV modelling framework applied to UG2 circuit data from Rustenburg Platinum Mines. Built XGBoost models with engineered lag features for predicting silica and iron concentrate grade. Implementing NMPC and MHE control systems using the do-mpc toolbox.

Research Paper XGBoost LOOCV Froth Flotation NMPC / MHE do-mpc
03 / App

MetaForge Assistant

Conceived and built a digital assistant for materials engineering students, integrating phase diagram visualisation, cooling curve simulation, and real-time data access via the Materials Project API. Built with Flask and React to democratise computational materials tools.

Full-Stack Flask React Materials Project API Phase Diagrams
04 / Research

Double Thermal Treatment: Rice Husk Silica Extraction

Conducted a proof-of-concept study on extracting high-purity amorphous silica from rice husks via a two-stage calcination process. Investigated applications in water purification, including microplastics removal, as part of the MICROCLEAN and AGRAVOLUTION initiatives.

Research Project Silica Extraction Calcination Waste Valorisation Water Purification
